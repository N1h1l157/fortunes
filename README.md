## mac 使用

```shell
# 安装
brew install fortune

# 添加字典
git clone git@github.com:n1h1l157/fortunes.git
strfile Mund4n3
mv Mund4n3 /opt/homebrew/Cellar/fortune/9708/share/games/fortunes

# 使用字典
fortune -e Mund4n3
```
